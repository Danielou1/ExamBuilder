Detailed Description

Service class responsible for exporting an model.Exam object to a Microsoft Word (.docx) document using Apache POI library. It handles the generation of exam papers, answer keys, and various formatting requirements including HTML content conversion, image embedding, and specific handling for different question types (MCQ, fill-in-the-blank, true/false).
