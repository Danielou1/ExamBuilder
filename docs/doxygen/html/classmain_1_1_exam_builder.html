Detailed Description

Main entry point for the ExamBuilder JavaFX application. This class extends javafx.application.Application and is responsible for initializing the primary stage, loading the main user interface from MainView.fxml, and setting up the core controller.

void main.ExamBuilder.start ( Stage primaryStage) throws Exception

The main entry point for the JavaFX application. This method is called after the application has been launched. It sets up the primary stage, loads the main user interface from MainView.fxml, links it with the controller.MainController, and displays the stage.
