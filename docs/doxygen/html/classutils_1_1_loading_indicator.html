Detailed Description

Utility class for displaying a modal loading indicator to the user. This indicator consists of a javafx.scene.control.ProgressIndicator and a "Loading..." label, shown in an undecorated, application-modal stage. It is used to provide visual feedback during long-running operations (e.g., export).
