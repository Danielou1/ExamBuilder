Detailed Description

Controller for the "Hinweise" (Instructions) dialog, which allows the user to configure general exam instructions, allowed aids (Hilfsmittel) using checkboxes and custom text, and the exam duration (Bearbeitungszeit). It updates the corresponding fields in the model.Exam object and provides a live preview of the generated instructions.
